# Euro area unemployment rate

Returns the monthly harmonised unemployment rate for the euro area from
the ECB's labour force statistics dataset (LFSI).

## Usage

``` r
ecb_unemployment(from = NULL, to = NULL, cache = TRUE)
```

## Arguments

- from:

  Optional start date.

- to:

  Optional end date.

- cache:

  Logical. Use cached data if available (default `TRUE`).

## Value

A data frame with columns:

- date:

  Date.

- value:

  Numeric. Unemployment rate as a percentage.
